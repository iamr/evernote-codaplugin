Evernote Wiki Plug-In for Coda 2

How does it work

This plug-in allows you to clip snippets and rendered HTML from Markdown to Evernote.

Intalling

Download, Unzip and Double click, that's all.

Shortcuts

^⌥⌘e
Clip file content
^⌥⇧⌘e
Clip selection
^⌥⌘m
Clip Markdown rendered file content
^⌥⇧⌘m
Clip Markdown rendered selection
